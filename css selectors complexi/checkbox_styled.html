CSS Diner

  • Selector de tip: A

    1) plate
    2) bento

  • ID-Selector: #ID

    3) #fancy

  • Selector descendent: AB

    4) plate apple

  • Combinatie descedent cu identificare: #id A

    5) #fancy pickle

  • Selector de clase: .className

    6) .small

  • Combinatie selector cu clase: A.className

    7) orange.small

  • Combinatie selector A cu selector cu clase:
    A>B.className

    8) bento>orange.small sau bento orange.small

  • Combinatie prin virgula: A, B, ..., div

    9) plate, bento, div

  • Combinatie elector universal *: A, B, *

    10) plate, bento, *

  • Selector universal * : A*

    11) plate *

  • Selectorul Sibling alăturat: A + B

    12) plate + apple

  • Selectorul general din Sibling : A ~ B
    Selectorul Adiacent (A + B),
    cu excepția faptului că primește toate
    elementele următoare în loc de unul.

    13) bento ~ pickle

  • Child Selector: A > B

    14) plate > apple

  • Pseudo selector primul copil :first-child

    15) orange:first-child

  • Pseudo selector singurul copil :only-child

    16) plate :only-child

  • Pseudo selector ultimul copil :last-child

    17) .small:last-child

  • Pseudo selector al N-lea copil A:nth-child(n)

    18) plate:nth-child(3)

  • Pseudo selector al N-lea pentru ultimul copil
    A:nth-last-child(n)

    19) bento:nth-last-child(3)

  • First of Type Selector (primul selector de tip)
    A:first-of-type

    20) apple:first-of-type

  • Nth of Type Selector (al N-lea selector de tip)
    :nth-of-type(even or odd) par-impar

    21) plate:nth-of-type(even)

  • Nth-of-type Selector with Formula :nth-of-type(An+B)
    fiecare al n-lea element, incepind cu elemntul n (An+B)

    22) plate:nth-of-type(2n+3)

  • Only of Type Selector (singurul element de tip)
    :only-of-type

    23) apple:only-of-type

  • Last of Type Selector (ultimul element de tip)
    :last-of-type

    24) .small:last-of-type

  • Empty Selector (selector-gol) :empty

    25) bento:empty

  • Negation Pseudo-class (negare) :not(X)

    26) apple:not(.small)

  • Selector [attribute]

    27) [for]

  • Attribute Selector A[attribute]

    28) plate[for]

  • Attribute Value Selector [attribute="value"]

    29) bento[for="Vitaly"]

  • Attribute Starts With Selector
    (Atributul începe cu selectorul) [attribute^="value"]

    30) [for^="Sa"]

  • Attribute Ends With Selector
    (Atributul se termina cu selectorul)
    [attribute$="value"]

    31) [for$="ato"]

  • Attribute Wildcard Selector
    (Atribut cu caractere specifice oriunde)
    [attribute*="value"]

    32) [for*="obb"]
